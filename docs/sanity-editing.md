# Sanity editing

## Projectfoto's aanpassen

1. Open Sanity Studio: `/studio`.
2. Ga naar `Project`.
3. Open het project dat je wilt aanpassen.
4. Vervang `Hoofdbeeld / cover`.
   - Dit beeld wordt gebruikt in overzichten en bovenaan de projectpagina.
5. Voeg beelden toe onder `Projectbeelden / gallery`.
   - Deze beelden verschijnen op de projectpagina.
   - Sleep beelden om de volgorde te wijzigen.
   - Gebruik `Bijschrift` optioneel voor interne context.
   - Vul `Alt-tekst` in als korte beschrijving van het beeld.
6. Klik `Publish`.
7. Refresh de site. De site gebruikt Sanity-beelden als ze bestaan en valt terug op de lokale beelden als een project nog geen beelden heeft.
