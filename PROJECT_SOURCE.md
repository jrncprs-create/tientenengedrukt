# Projectbron — Joep / tientenengedrukt

## Harde projectregel

Deze site draait als custom portfolio-site via **Next.js** met **Sanity CMS**.

Gebruik **niet** meer WordPress of Divi als uitgangspunt voor dit project.

## Stack

- **Frontend / site:** Next.js, App Router, TypeScript
- **CMS:** Sanity
- **Hosting site:** Vercel
- **Code:** GitHub-repo `jrncprs-create/tientenengedrukt`
- **Domein/DNS/mail:** Cloud86 waar relevant, maar niet als sitebuilder/CMS-route
- **Werktool:** Cursor / VS Code voor code-aanpassingen

## Werkwijze

1. Ontwerp, interactie en componenten worden als custom Next.js-site gebouwd.
2. Projecten, teksten, beelden en portfolio-content worden via Sanity beheerd.
3. GitHub is de bron voor code en projectafspraken.
4. Vercel deployt de website.
5. Cloud86 blijft alleen relevant voor domein, DNS en/of mail-instellingen.

## HKU-aanmeldportfolio subdomein

Joep wil tijdelijk of apart een HKU-aanmeldportfolio publiceren op een subdomein van `tientenengedrukt.nl`.

Doel:

- portfolio tonen voor toelating HKU Associate degree Digital Media;
- nadruk op proces, onderzoek, schetsen, keuzes, iteraties en onaf werk;
- vormgeving moet aansluiten op `tientenengedrukt`, maar de HKU-route mag inhoudelijk duidelijker en toelatingsgerichter zijn;
- aangeleverde mapstructuur moet intact blijven;
- afbeeldingen binnen mappen blijven lineair geordend van `1` tot `x`, niet visueel door elkaar gemixt.

## HKU-inhoudelijke focus

De HKU Ad Digital Media kijkt o.a. naar:

- conceptueel vermogen;
- eigenheid en diversiteit;
- algemeen beeldend vermogen;
- passende vaardigheden en technisch vermogen;
- communicatieve en reflectieve vermogens;
- affiniteit met het vakgebied;
- culturele en maatschappelijke belangstelling.

Voor dit portfolio betekent dat:

- eindbeelden tonen, maar proces minimaal even belangrijk maken;
- per project laten zien: vraag/context, Joep's rol, onderzoek/proces, keuzes, eindresultaat, reflectie;
- onaf werk en schetsen niet verstoppen, maar bewust presenteren als bewijs van denken en maken.
